### IPL Match Win Probability Predictor
**Technologies:** Python, Pandas, NumPy, Scikit-learn, Matplotlib  

- Developed a machine learning model to predict the probability of winning an IPL match during the second innings based on the current match situation.  
- Applied feature engineering and trained a **Logistic Regression model** using historical IPL data (2008–2025).  
- Utilized **Pandas and NumPy** for data preprocessing and **Scikit-learn** for model training.  
- Implemented visualizations using **Matplotlib** to analyze match dynamics and prediction outcomes.  
- Achieved approximately **81% prediction accuracy** using an 80–20 train–test split.
